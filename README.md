# PlanPal

PlanPal is a simple schedule planning app that helps students organize their classes, tasks, and goals in one place.

# What it does
You can create a weekly schedule, add your tasks or lectures, set reminders, and see everything in one clear view.

# Why it matters
Many students forget tasks or miss deadlines because they have too many things to remember. PlanPal helps you stay on track and reduce stress.

# Who it’s for
- Students in school or university  
- Anyone who wants to manage time better  

# Key features
- Add and edit your daily schedule  
- Get reminders before each task  
- Weekly and daily calendar views  
- Simple and clean design  

# Technology
PlanPal will be built with React (for the front) and Node.js (for the back) — simple web tools that make it fast and easy to use.
